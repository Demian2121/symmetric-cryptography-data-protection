# Aplicación de Criptografía Simétrica para Protección de Datos

## Descripción

Proyecto enfocado en la implementación de un mecanismo de criptografía simétrica para la protección de información. Se desarrolló un sistema capaz de encriptar y desencriptar datos utilizando claves secretas, garantizando la confidencialidad de la información.

## Objetivo

Implementar un sistema básico de cifrado simétrico en Python que permita proteger datos sensibles mediante procesos de encriptación y posterior recuperación segura de la información.

## Tecnologías utilizadas

- Python
- Librerías de criptografía (ej. cryptography o similares)

## Funcionalidades

1. Generación de clave secreta.
2. Encriptación de datos.
3. Desencriptación de datos.
4. Validación de integridad del proceso.

## Metodología

- Implementación de algoritmo de cifrado simétrico.
- Uso de claves compartidas para encriptación y desencriptación.
- Pruebas de funcionamiento con diferentes tipos de datos.
- Verificación de correcta recuperación de la información original.

## Conceptos aplicados

- Criptografía simétrica
- Seguridad de la información
- Protección de datos
- Confidencialidad
- Gestión de claves
